# Práctica 2: Cifrado de Vigenere

## Introducción

Cifrado y descifrado de mensajes mediante el cifrado de Vigenere.

## Ejecución

~~~
    py vigenere.py "'mensaje'" "'clave'"
~~~

Ejemplo de ejecución:

~~~
    py vigenere.py "este mensaje se autodestruira" "mision"
~~~
